# ServerManager - 聊天室管理端

基于 WPF (.NET 8) 的聊天室后台管理工具，提供在线用户管理、IP 封禁、播放列表管理、公告发布、清空消息等功能。

## 技术栈
- .NET 8 WPF
- SignalR Client
- REST API 调用

## 功能特性
- 管理员登录（仅 admin 账号可连接管理 Hub）
- 在线用户列表（实时刷新，可踢出、禁言、封禁/解封）
- 封禁用户列表（全局查看，一键解封）
- IP 黑名单管理（手动添加/解封，封禁用户时自动获取 IP）
- 播放列表查看与删除
- 清空所有聊天消息
- 公告发布与清除
- 服务器日志查看

## 环境要求
- Windows 10/11 或 Windows Server 2016+
- .NET 8 Desktop Runtime (或自包含发布)
